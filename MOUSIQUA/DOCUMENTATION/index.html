Description

MOUSIQUA is a modern music Band and Musician Template, it allows bands and musician to introduce themself with a unique simple scrollable one page fully loaded with a lot of section to get people know a lot of of infos.

 

The template have :

  1. Responsive layout
  2. Retina Ready
  3. Multiple section to feat every needs
  4. One page navigation
  5. Html 5 and CSS3
  6. Unique design
  7. Block design sections
  8. Simple to use audio player
  9. Single album page
  10. Build with the latest bootstrap v4
  11. Animated twitter feed
  12. and much more...

Twitter setup

MOUSIQUA  come with twitter feed section to display latest tweets in  your  site, by default this area in the footer display a default tweets for demo purposes, but of cours you can display your own twitter feed.

To change the twitter widget located in the footer to display your own tweets, simply open script.js file located in js folder inside the main HTML folder then locate the twitter feed setup section  look for the line of codes that setup the twitter feed then replace the profile name  by yours.

More info about instgram feed option can be found here :

https://github.com/jasonmayes/Twitter-Post-Fetcher

Audio player

MOUSIQUA uses audio.js a drop-in javascript library that allows HTML5’s <audio> tag to be used anywhere.It uses native <audio> where available and an invisible flash player to emulate <audio> for other browsers. It provides a consistent html player UI to all browsers which can be styled used standard css.

The player can be customized and easly add trakc to it by simply adding the url to the audio that you want to play.

 

More info about audio.js in the link below :

 

https://kolber.github.io/audiojs/

Counter setup

MOUSIQUA come with a counter to show how much time is left until the next band or musician show time. The setup is very easy just open script.js and scroll down until you find the countdown section you will find the lines below just change the date that you want in the function that handel to countdown.

 

  1. // Count down setup
  2.  
  3. $('.countdown').countdown('2017/6/10', function(event) {
  4. $(this).html(event.strftime('%D days %H:%M:%S'));
  5. });

 

For more info just visit the link below :

http://hilios.github.io/jQuery.countdown/
